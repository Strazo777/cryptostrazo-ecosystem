# CryptoStrazo Client Plugins

A collection of client-side plugins and integration packages for **CryptoStrazo**.
This repository is a monorepo that will contain multiple independent integrations for different platforms and stacks.

The goal is to provide ready-to-use components that help merchants connect their systems to CryptoStrazo with a consistent approach to security, reliability, and minimal overhead.

## Contents

Each plugin lives in its own folder and is maintained as a standalone integration.
Documentation and implementation details are kept inside the corresponding plugin directory or in external docs.
